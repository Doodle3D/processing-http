Library
Name

JSONObject
Description A JSONObject represents some data from a JSON response. You can query the type of data and access its value(s) using the functions in this class.
Methods
booleanValue() Returns the value of this JSONObject as a boolean.
doubleValue() Returns the value of this JSONObject as a double.
floatValue() Returns the value of this JSONObject as a float.
get() Returns the JSONObject value for the specified key, given that this JSONObject is a dictionary.
intValue() Returns the value of this JSONObject as an int.
isBoolean() Returns true if this JSONObject represents a boolean value, false otherwise.
isFloatingPoint() Returns true if this JSONObject represents a floating-point value, false otherwise.
isInteger() Returns true if this JSONObject represents an integer value, false otherwise.
isList() Returns true if this JSONObject represents a list whose elements can be retrieved by index position.
isMap() Returns true if this JSONObject represents a Map whose values can be retrieved by key.
isNull() Returns true if this JSONObject represents a null value.
isString() Returns true if this JSONObject represents a String, false otherwise.
longValue() Returns the value of this JSONObject as a long.
size() Returns the number of JSONObject elements in this list, given that this JSONObject represents a list.
stringValue() Returns the value of this JSONObject as a String.
toString() Returns the textual representation of this JSON object.
