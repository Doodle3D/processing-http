Library
Class

JSONObject
Name

get
Description Returns the JSONObject value for the specified key, given that this JSONObject is a dictionary.
Syntax 
jsonObject.get(key)
jsonObject.get(i)
Parameters
jsonObject JSONObject: any variable of type JSONObject
key The key you wish to retrieve the value for
i Index of the element you wish to retrieve
Returns JSONObject
